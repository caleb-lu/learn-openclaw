# Tools - Helper

## Available Tools

### calendar
Manage calendar events (create, read, update, delete).
- **Usage**: When the user wants to schedule, check, or modify calendar events.
- **Example**: "Schedule a team meeting for tomorrow at 2pm."

### weather
Get current weather conditions and forecasts.
- **Usage**: When the user asks about weather for planning purposes.
- **Example**: "Will it rain this weekend?"

### reminder
Set and manage reminders for tasks and events.
- **Usage**: When the user wants to be reminded of something at a specific time.
- **Example**: "Remind me to call the dentist tomorrow at 10am."

### email
Read, compose, and manage emails (requires email integration).
- **Usage**: When the user wants to check, send, or organize emails.
- **Example**: "Show me unread emails from this morning."

### web_search
Search the web for information.
- **Usage**: When the user needs up-to-date information on any topic.
- **Example**: "Find the best coffee shops near my location."

### calculator
Perform mathematical calculations.
- **Usage**: When the user needs help with budgeting, tips, conversions, or any math.
- **Example**: "Split a $120 bill 3 ways with 18% tip."

## Tool Usage Guidelines
- Always confirm before creating calendar events or sending emails.
- Use reminders proactively when the user mentions time-sensitive tasks.
- Combine tools when appropriate (e.g., check weather before scheduling an outdoor event).
