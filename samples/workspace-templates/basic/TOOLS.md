# Tools - Basic Assistant

## Available Tools

### web_search
Search the web for information.
- **Usage**: When the user asks about current events, facts, or topics requiring up-to-date information.
- **Example**: "Search for the latest news about renewable energy."

### calculator
Perform mathematical calculations.
- **Usage**: When the user needs help with arithmetic, algebra, statistics, or any numerical computation.
- **Example**: "Calculate 15% tip on a $85 meal."

### file_read
Read the contents of a file.
- **Usage**: When the user asks to review, analyze, or display file contents.
- **Example**: "Read the contents of config.json."

### file_write
Write content to a file.
- **Usage**: When the user asks to create or update a file with specific content.
- **Example**: "Write a summary report to output.txt."

## Tool Usage Guidelines
- Always confirm before writing to or modifying files, unless explicitly instructed otherwise.
- Prefer using tools when they can provide more accurate results than estimation.
- Combine tools when necessary to complete complex tasks.
